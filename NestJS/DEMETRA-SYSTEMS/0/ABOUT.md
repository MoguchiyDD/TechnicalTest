## New Commands

```Bash
# Migrate Deploy
$ yarn prisma:db:deploy

# Up Docker Compose
$ yarn db:up

# Remove Docker Compose
$ yarn db:rm

# Restart Docker Compose (db:rm + db:up + sleep + migrate deploy)
$ yarn db:reboot
```
